# StudentAid AI – Homework Solver

**StudentAid AI** is a web application that helps students solve homework problems from PDFs or images. Upload your homework file, and the app uses **Google Gemini AI** to provide clear, step-by-step solutions.  

Built with **Streamlit**, **PyMuPDF**, and **Google Gemini API**.

---

## Features

- Upload homework files in **PDF** or **image** formats (`PNG`, `JPG`, `JPEG`).  
- Automatically extract text from PDFs using **PyMuPDF**.  
- Generate step-by-step solutions using **Google Gemini 2.5-Flash**.  
- Analyze non-text PDFs/images using **vision-based AI analysis**.  

---

## Tech Stack

- **Python 3.10+**  
- **Streamlit** – interactive web UI  
- **PyMuPDF (fitz)** – PDF text extraction  
- **Pillow (PIL)** – image handling  
- **Google Gemini API** – AI model for homework solving  
- **Docker** – optional containerized deployment

## Setup Instructions

### 1. Clone the Project

```bash
git clone https://github.com/bushraqurabi/StudentAid_AI.git
cd StudentAid_AI
````

### 2. Set Up Virtual Environment

```bash
# Create virtual environment
python -m venv venv

# Activate on Windows
venv\Scripts\activate

# Activate on macOS/Linux
source venv/bin/activate
```

### 3. Install Dependencies

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

**Required Packages** (`requirements.txt`):

```
streamlit
python-dotenv
PyMuPDF
Pillow
google-generativeai
```

### 4. Configure Environment Variables

Create a `.env` file in the project root:

```
GEMINI_API_KEY=your_gemini_api_key_here
```

> Get your Gemini API key here: [Google AI Studio API Keys](https://aistudio.google.com/app/api-keys)

**Important:** Do **not** commit your `.env` file to GitHub. Ensure `.env` and your virtual environment are added to `.gitignore`:

```
# .gitignore
.env
venv/
__pycache__/
```

### 5. Run the App Locally

```bash
streamlit run app.py
```

Open the app in your browser, upload your homework PDF or image, and click **Solve Homework**.

## Docker Setup (Optional)

Run the app in a Docker container for easier deployment.

1. **Build Docker Image**

```bash
docker build -t studentaid-ai .
```

2. **Run Docker Container**

```bash
docker run -p 8501:8501 --env GEMINI_API_KEY=your_gemini_api_key_here studentaid-ai
```

Access the app at [http://localhost:8501](http://localhost:8501).

> Make sure to replace `your_gemini_api_key_here` with your actual API key.
